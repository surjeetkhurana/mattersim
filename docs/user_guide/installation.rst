
Installation
============

Install from source code
------------------------

Requirements:

* Python == 3.9
* PyTorch == 1.11.0

To install the package, run the following command under the root of the folder:

.. code-block:: console

    conda env create -f environment.yaml
    conda activate mattersim
    pip install -e .
